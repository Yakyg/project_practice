Разработка Телеграм-бота SafetyBot на Python

Реализация помощника по безопасности с использованием Python и Telegram Bot API.

Основные этапы

  • Начальная разработка бота с помощью Python и Telegram Bot API
  • Создание интерактивного меню с inline-кнопками
  • Интеграция JSON-базы данных с вопросами/ответами для викторины, первой помощью и ежедневных напоминаний
  • Реализация Docker-контейнеризации для стабильного развёртывания решения

Описание и функционал Safety_aid_Bot

Телеграм-бот Safety_aid_bot — это бот-помощник по безопасности, предоставляющий информацию об оказании первой помощи, ежедневно напоминает о правилах безопасности и даёт возможность пройти викторину по правилам безопасности.

Как работает

  • Пользователь выбирает функцию (викторина или оказание первой помощи), а бот, в соответствии с выбором, проводит викторину либо даёт выбрать направление оказания первой помощи (например, ПП при ожоге). После чего бот даёт инструкцию.

Преимущества

  • Лёгкий доступ к информации
  • Возможность быстро получить нужную инструкцию в экстренной ситуации

Функционал

  • Кнопки:
    • Викторина — сыграть в викторину о правилах безопасности.
    • Советы по первой помощи — получить информацию о необходимых действиях при оказании первой помощи.
    • Описание бота — получить описание возможностей бота.
  • Локальная база в формате JSON
